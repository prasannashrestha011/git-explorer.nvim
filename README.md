
# git-explorer.nvim

A simple Neovim floating Git Explorer modal.

## Features
- Shows Staged / Unstaged / Untracked files
- Git status icons per section
- File type icons for files
- Open file by pressing `<CR>`
- Toggle modal with `<leader>ge`

### Using packer.nvim
